# @your-username/react-video-player

Production-grade React video player with YouTube-style features.

## Features

- ✅ Thumbnail preview on hover
- ✅ Background prefetch when paused
- ✅ HLS streaming support
- ✅ Keyboard shortcuts
- ✅ Picture-in-Picture
- ✅ TypeScript support
- ✅ Next.js compatible

## Installation

```bash
npm install @your-username/react-video-player
```

## Quick Start

```tsx
import { VideoPlayer } from "@sanishmdhr/react-video-player";
import "@sanishmdhr/react-video-player/styles";

function App() {
  return (
    <VideoPlayer src="video.mp4" enablePreview={true} enablePrefetch={true} />
  );
}
```
